Projects

Remote-controlled thermostat

I developed the thermostat using an ESP32 microcontroller, which received temperature data from a connected sensor. The system was remotely controlled via a mobile application, allowing users to monitor and adjust settings in real time.

System for Measuring Particulate Matter (PM)

I developed a particulate matter monitoring system using an ESP32 microcontroller and an SDS011 dust sensor. The ESP32 collected real-time PM2.5 and PM10 data from the sensor and stored it in an SQL database. I also designed and built a custom website to display the collected data, allowing users to monitor air quality remotely and in real time. This project involved full-stack development: hardware integration, data acquisition, database management, and front-end web development. I plan to add this project to this website in the future, including detailed information and visuals.
